# ChatGPT Business Landing (React + TypeScript)

Single‑page landing for selling ChatGPT Business access for 1 month at $20.

## Files
- `index.html` — base version.
- `index-sales.html` — sales version with more aggressive copy.
- `src/App.tsx` — markup and logic.
- `src/styles.css` — styling.
- `DOCUMENTATION.md` — documentation (RU).
- `DOCUMENTATION.ru.md` — documentation (RU duplicate).
- `DOCUMENTATION.en.md` — documentation (EN).
- `FAQ.md` — FAQ/policies (RU).
- `FAQ.ru.md` — FAQ/policies (RU duplicate).
- `FAQ.en.md` — FAQ/policies (EN).

## Quick start
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
